Seletor Universal

Seleciona todos os elementos da página.

* {
   margin: 0;
   padding: 0;
}

Seletor de Classe

Seleciona elementos através de uma classe.

.botao {
   background: blue;
}

Seletor de ID

Seleciona um elemento único.

#titulo {
   color: red;
}

Combinadores

Permitem selecionar elementos dentro de outros elementos.

nav a {
   text-decoration: none;
}

Pseudo-classes

Aplicam estilos em estados específicos.

button:hover {
   background: green;
}

Pseudo-elementos

Permitem estilizar partes específicas de um elemento.

p::first-letter {
   font-size: 2rem;
}

Especificidade CSS

A especificidade define qual regra CSS terá prioridade quando houver conflito entre seletores.
